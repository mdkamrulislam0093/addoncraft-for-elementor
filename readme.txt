== Installation ==

Note : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.

1. install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. All available elements are accessible within the Elementor editor interface.
4. All these elements are organized under the 'AddonCraft' category in your Elementor widget panel for easy access.

== Frequently Asked Questions ==

= Can I use the plugin without Elementor Page Builder? =
This is specifically an Elementor add-on, so it requires Elementor to function properly. It cannot be used as a standalone plugin.

= Does it work with any theme? =
Absolutely! It will work with any theme where Elementor works.

= Do i need Elementor Pro? =
This add-on works with the free version of Elementor - the premium version isn't required.
